### 入力おたすけ

-   `rfce + Tab` : 関数コンポーネント作成

```javascript
import React from 'react'

function Test() {
	return <div>Test</div>
}

export default Test
```

-   `rafce + Tab` : allow 関数コンポーネント作成

```javascript
import React from 'react'

const Test = () => {
	return <div>Test</div>
}

export default Test
```

-   `div.className + Tab` : className つきの div 作成
